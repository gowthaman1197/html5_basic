What's HTML?

Hypertext Markup Language.

Structure of webpages.

Elements are building block of HTML pages and represented by tags.

Browsers do not display the HTML tags. They use them to render the content of the tags.



Explanation

Doctype - this document to be HTML5

html - root element of an HTML page.

head - contains meta information about the document

title - specify the title of document.

body- visible page of content

h1 - large heading

p - define paragraph
